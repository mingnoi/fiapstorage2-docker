# fiapstorage2-docker

 - FIAPStorage2 (IEEE1888) Dockerfile
 - Endpoint: http://(docker host address):8080/axis2/services/FIAPStorage

## Start containers

```bash
docker-compose.exe up -d
```

## Stop containers

```bash
docker-compose.exe stop
```

## Remove containers

```bash
docker-compose.exe down
```
